Spacing

Elements need to have space between them to breath. Showing a block of text that is trying and convert a visitor therefore follows the principle of Legible as fuck and Don't Fear the Whitespace.

Whitespace

Whitespace is an active element in design that provides breathing room, helping to guide the viewerâ€™s focus and emphasise key content.

By balancing negative space with visual elements, it enhances readability, creates structure, and fosters a clean, professional aesthetic.

The problem this is solving is that when trying to design a layout for a landing page, there is a tendency to cram in elements without meaning and when stepping back it viewed as a whole without the question of "is the attention of the visitor at each section ensured".

Core Concept

The Entrepreneurship System sets landing page text to have no padding and no margin (long-form will have though) but instead borrows from Flutter to use sized boxes to space out elements.

This is because, controlling the space with margins can become cumbersome when it comes making it responsive.

Basically if you set a margin to change the space, you end up writing a ton of class for each element and section.

Sized Boxes

So instead, a Sized Box is placed to create space, and because it is calculated from the same scaling factor for font sizing and viewport width, the spacing is proportionate and makes sense over various devices.

Usage

Sized boxes can be used in anyway needed. But as a bit of guidance:

Sized Box Small

For a bit of a gap between large titles and paragraph text. It will be nearly nothing on mobile and a bit of room for desktop.

Sized Box Medium

To space medium sized titles with a 5-10 words from paragraph text.

Sized Box Large

Used to space two different elements from each other, like an image and a title.

Sized Box Massive

Basically for upper and lower padding for entire sections.
